# Wind Simulation System

## Hardware

- Arduino Pro Micro
- 2x Thermalright TL-C12C PWM Fans
- 12V 1A Power Supply

## SimHub Configuration

Device Type:
- PWM Fan Controller

Pins:
- Fan 1 PWM = D9
- Fan 2 PWM = D10

## Features

- Speed-based airflow
- Left/right airflow effects
- Drafting effects

## Credits

## Notes

Firmware is generated and uploaded directly by SimHub.
No custom Arduino code is required.
